CARACTERISTICAS

•linguagem interpretada: escreve o programa, salvo no arq .java, o compilador e transforma em .class e sera interpretado em linguagem de maquina para entender.

•é diferente das compilada, ela é compilada para um bytecode (.class) e interpretada por uma maquina virtual (.jvm)

•a linguagem java não é a única da plataforma java, sendo a segunda criada.

• ela tem uma maquina virtual que emula, assim não deixando preso a um S.O ou um hardware "escreva uma vez e execute em qualquer lugar"

tipos: java SE, java EE, java ME e java fx

FASE DE EXECUÇÃO

1 - codigo-fonte (.java)

2 - JDK para copilar e gerar um bytecode(.class)

3 - jvm lê o arquivo .class e as bibliotecas padrões do java que estão no JRE

jvm > jre > jdk
